## ðŸ”§ Customization

### Change App Name
Edit `vite.config.js`:
```javascript
manifest: {
  name: 'Your App Name',
  short_name: 'App',
  // ...
}
```

### Change Theme Color
Edit `vite.config.js`:
```javascript
manifest: {
  theme_color: '#your-color',
  background_color: '#your-bg-color',
}
```

### Change Icons
1. Replace `public/mask-icon.svg`
2. Run `npm run generate-icons`
3. Rebuild: `npm run build`

### Adjust Caching
Edit `vite.config.js` â†’ `workbox` section:
```javascript
runtimeCaching: [
  {
    urlPattern: /your-pattern/,
    handler: 'CacheFirst', // or 'NetworkFirst', 'StaleWhileRevalidate'
    options: {
      cacheName: 'your-cache',
      expiration: { maxAgeSeconds: 3600 }
    }
  }
]
```
